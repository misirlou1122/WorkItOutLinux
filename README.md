# Work It Out: Linux

A Linux final study app built as a single-page static web app.

The app includes Linux lessons, flash cards, daily drills, command practice, missed-question review, final practice, and English/Spanish study modes.

## Open the app

Open `index.html` in a browser.

No build step is required. The app uses plain HTML, CSS, and JavaScript.

## Features

- Topic-based Linux review lessons
- Flash cards and saved cards
- Daily drills and final practice quizzes
- Missed-question notebook
- Command Lab for practical command memorization
- English and Spanish modes
- Offline-friendly app files with a service worker when served over HTTP

## Study Content

The study material was built from Linux final review topics and expanded with general Linux, Red Hat Enterprise Linux, GNU, and Linux manual page concepts.

Do not upload private class documents, instructor files, or copyrighted course materials unless you have permission to share them.

## Project Structure

- `index.html` - App entry point
- `styles.css` - Visual design and layout
- `app.js` - Lessons, quizzes, flash cards, and app behavior
- `manifest.webmanifest` - Installable app metadata
- `service-worker.js` - Offline cache support for HTTP hosting
- `assets/` - App icons and background image
